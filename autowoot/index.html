Plug.dj AutoWoot

This is a basic, lightweight script that will woot for you whenever a new song starts playing.

While the script is running, an extra 'W' icon will be added to the top of your chat, if you click this it will toggle the autowoot. Green means you're currently autowooting, red means you're not.

Usage

Follow these instructions to begin using the autowooter: 

1) Create a bookmark by dragging the kitteh to your bookmark bar.
2) Go to your plug.dj room.
3) Click the bookmark to start autowooting.
